# SentinelAI — AI-Powered Cyber Defense Platform

SentinelAI is a professional, local-first, AI-powered cyber-defense platform and SOC simulation system.

It integrates defensive host monitoring, Windows Event Log collection, interactive honeypot systems, MITRE ATT&CK mapping, real-time attack visualization, dynamic reporting, and local AI collaboration through Ollama.

## Current Phase

Phase 0 — Repository and documentation foundation.
