# 16 — MITRE ATT&CK


## Purpose
Map local attack events and suspicious Windows logs to ATT&CK tactics and techniques.

## Required sections
- Tactic columns or matrix
- Search
- Tactic filter
- Technique cards
- Local event counts
- Confidence
- Detail drawer
- Related attacks/logs
- Detection and mitigation notes

## Acceptance
Mappings cite stable technique IDs and distinguish rule-based from AI-suggested mappings.


## Architecture rule

Create the page folder only when this module is implemented. Keep page-specific components, hooks, services, utilities and assets inside that folder. Shared primitives remain in the shared component library.
