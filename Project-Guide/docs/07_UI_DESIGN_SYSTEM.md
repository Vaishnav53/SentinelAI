# 07 — UI Design System

## Visual language

SentinelAI uses a dense enterprise SOC visual language:

- Deep navy background
- Translucent elevated surfaces
- Cyan primary accents
- Purple for AI
- Green for healthy states
- Yellow/orange for warning
- Red for critical
- Subtle grid and particle backgrounds
- Compact typography
- Restrained glow

## Spacing

Use an 8px base:

- 4px micro
- 8px small
- 12px compact
- 16px standard
- 24px section
- 32px major

## Radius

- 6px small controls
- 8px compact cards
- 12px standard cards
- 16px hero surfaces
- Full radius for badges

## Typography

Recommended:

- UI: Inter
- Technical labels and identifiers: JetBrains Mono

Hierarchy:

- Page title: 20–28px
- Card title: 12–15px
- Metric value: 20–32px
- Body: 11–14px
- Labels: 9–11px uppercase

## Motion

- 150–250ms hover/selection
- 300–500ms panel transitions
- 1.5–3s status pulses
- Radar loops 5–10s
- Never animate large surfaces continuously at high intensity

## Layout

- Sidebar fixed
- Header compact
- Main content scrolls
- Cards align to grid
- Avoid nested scrollbars unless needed for tables or code panes
